# ChatGPT Prompt Engineering for Developers

## By Andrew Ng DeepLearning.AI

### Types of LLMS

- Base LLM
- Instruction-tuned LLM

### Principles of Prompt Engineering

- Write clear and specific instructions.
- Give the model time to think.
- Use delimiters to clearly indicate different parts of the input.
- Give the model examples of the input and output.
- Specify the steps required to complete the task.
- Specify the format of the output.
- Ask the model to work out its own solution before rushing to a conclusion.
- Ask model to first find relevant information, then answer the question.
