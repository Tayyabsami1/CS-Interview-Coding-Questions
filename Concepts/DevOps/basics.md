## Nginx and Reverse Proxy

- Nginx (pronounced "Engine-X") is software that sits in front of your web applications and handles incoming requests.
  -It is commonly used as:
- 🌐 Web server
- 🔄 Reverse proxy
- ⚖️ Load balancer
- 🚀 Cache server

- Nginx is a fast web server and reverse proxy that receives web requests, serves static content when it can, and forwards other requests to the correct backend application or server.

### Why developers love Nginx

- 🚀 Very fast
- 💾 Uses little memory
- 🔒 Improves security
- ⚖️ Balances traffic across servers
- 📂 Serves images, CSS, and JavaScript efficiently
- 🌍 Can handle thousands of simultaneous connections

### Reverse Proxy

- A reverse proxy is a smart receptionist that stands in front of your servers, receives every request first, and decides where it should go while also protecting, speeding up, and hiding the actual servers.

### Forward Proxy

- A forward proxy sits infront of you so it basically hides you.
